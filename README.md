## Description

Unlimited nested category using graphql with mongodb and redis for caching.

## NestJS install

```bash
$ npm i -g @nestjs/cli
```


## Project Clone

```bash
$ git clone https://github.com/ank-p-turza/category-gql.git

# go to project directory
$ cd category-gql
```

## Mongo and Redis setup

```bash
$ docker compose up -d
```


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Usage
For local UI access go to 
```bash
http://localhost:3000/graphql
```
or access it on postman.
